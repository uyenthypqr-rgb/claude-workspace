## 6. ELEMENT VISUAL CHI TIẾT

### Doodle accents (subtle, not overload):
- **1 ngôi sao 4 cánh** ★ gold size 16px gần "168" để add charm
- **1 đường wave curve** mỏng gold (1.5pt) phía trên CTA strip
- **3 small icons** trong cards: ❤️ teal (Card 1), ⚔️ gold (Card 2), 🏆 rose (Card 3) — vector flat outline style, NOT 3D emoji
  - Hoặc dùng icon thay thế: 👁️ (thấu cảm) / 🛡️ (bản lĩnh) / 🌟 (thành quả)

### Card style:
- Background trắng `#FFFFFF` opacity 60% trên nền cream
- Border: 1px solid gold `#B99A44` opacity 30%
- Border radius: 12px
- Padding: 20px
- Drop shadow nhẹ: `0 2px 8px rgba(185,154,68,0.15)`

### CTA button:
- Solid gold `#B99A44` background
- Text: white `#FFFFFF` Montserrat Bold uppercase
- Border radius: 8px
- Padding: 14px 32px
- Drop shadow: `0 4px 12px rgba(185,154,68,0.3)`

### Big number "168":
- Font: Playfair Display Bold 220pt
- Fill: Linear gradient gold (#DDC96C → #BA9B44 → #966C26) 135deg
- Optional: drop shadow `0 6px 16px rgba(150,108,38,0.3)` để add depth

## 9. TIPS DESIGN TRONG CANVA / FIGMA

**Canva:**
- Search template: "Vietnam real estate post" hoặc "luxury training poster" để có base
- Set canvas custom 1080×1350
- Brand kit: tạo brand kit PQR với 3 màu primary (cream + gold + navy)
- Font: Cài Playfair Display + Allura free từ Canva (đã có sẵn)
- Logo: upload `Logo-PQR-Final-2.png` từ Drive

**Figma:**
- Frame 1080×1350
- Local styles: tạo color tokens (cream-bg, gold-primary, navy-title, etc.)
- Typography: setup text styles trước cho 8 cấp font ở section #3
- Plugin gợi ý: "Iconify" cho icon flat, "Unsplash" — CẤM dùng cho stock generic

**Photoshop / Illustrator:**
- Document 1080×1350px 72dpi sRGB
- Smart object cho logo để scale không vỡ
- Layer style: Gradient Overlay cho big number 168
